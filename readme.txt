Create text shortocodes with ease!

== Description ==

This plugin allows for the creation of text shortcodes that could be used across you site. The main advantage is that information that occurs throughout the site ( like address ) could be encapsulated in a shortcode and easily edited from one place only.

== Installation ==

1. Install Simple Text Shortcodes either via the WordPress.org plugin directory, or by uploading the files to your server.
1. Activate the plugin.
1. Create some posts in the Shortcodes post type.
1. Use the shortcodes :)
